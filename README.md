# Personal Data Footprint Tracker

An end-to-end data engineering and analytics project that collects, processes, and analyzes personal digital data from multiple sources to generate meaningful insights and predictions.

## Project Overview

This project aims to build a centralized platform for managing and analyzing personal data from different sources such as:

- Bank statements
- Spotify listening history
- Google Calendar
- Screen time data
- Weather data (optional)
- Google Takeout (future)

The data is extracted, transformed, and loaded (ETL) into a PostgreSQL database, where it can be queried, analyzed, and visualized through a Flask web application.

## Objectives

- Build automated ETL pipelines
- Practice data engineering concepts
- Design a relational database
- Perform exploratory data analysis
- Develop predictive machine learning models
- Deploy the application using Docker
